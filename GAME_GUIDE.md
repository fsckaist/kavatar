# 넙죽이와 포닉스의 역습 - 게임 매뉴얼

본 가이드는 이미 룰을 숙지한 플레이어가 게임을 원활하게 조작하기 위한 **실행 중심 가이드**입니다.

-------------------------------------------------------

[AP (행동력) 획득 시스템]

모든 행동에는 AP가 소모됩니다. 매 턴 시작 시 다음과 같이 AP가 충전됩니다.

**기본 지급: +3 AP**
**영토 보너스: 내 땅 5칸마다 +1 AP 추가**
(예: 땅이 12칸이면, 기본 3 + 보너스 2 = 총 5 AP 획득)

-------------------------------------------------------

[버튼별 실행 방법]

각 버튼을 누르면 어떤 순서로 클릭해야 하는지 설명합니다.

1. 징집 (비용: 1 AP)
- 실행 방법:
  1) [징집] 버튼을 누른다. (또는 내 땅을 먼저 선택)
  2) 병력을 늘리고 싶은 **내 땅**을 클릭한다.
- 효과: 선택한 땅의 숫자가 +1 증가한다. (최대 5까지)

2. 요새화 (비용: 3 AP)
- 실행 방법:
  1) [요새화] 버튼을 누른다.
  2) 방어하고 싶은 **내 땅**을 클릭한다.
- 효과: 해당 땅에 하늘색 테두리가 생기며, 적의 공격을 1회 무효화한다.

3. 공격 및 점령 (비용: 2 AP)
- 실행 방법:
  1) **공격을 출발할 내 땅**을 먼저 클릭한다. (선택됨 표시)
  2) **공격하고 싶은 인접한 땅** (적 또는 중립)을 클릭한다.
  3) 내 병력이 상대보다 많으면 자동으로 점령된다.
- 주의사항: 병력 차이가 없다면 공격이 실패하고 AP만 소모되니 주의! (중립 땅은 내 병력이 2 이상이어야 점령 가능)

4. 정화 (비용: 2 AP) *16라운드 이후 활성화*
- 실행 방법:
  1) **내 땅**을 먼저 클릭한다.
  2) [정화] 버튼을 누른다.
  3) 인접한 **포닉스 땅(빨간색)**을 클릭한다.
- 효과: 포닉스 땅이 중립(회색)으로 변하며 무력화된다.

-------------------------------------------------------

[유틸리티 버튼]

- 되돌리기: 방금 한 행동을 취소하고 AP를 돌려받는다. (턴 종료 전이면 여러 번 가능)
- 턴 종료: 내 차례를 마치고 AP를 남긴 채 다음 팀으로 넘긴다. (종료 확인 창이 뜹니다)
